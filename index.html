The path to AI-ready

Most problems that arrive as AI requests are actually an earlier type in disguise. Work through each type in order — resolving an earlier type often eliminates the need to go further.

1
Resolve first

Ownership problem

No one is clearly accountable. Tools don't fill ownership gaps — they make them more complicated. Name one owner before anything else.

2
Then check

Policy problem

The rules are unclear or missing. Automating ambiguity doesn't resolve it — it just makes it faster. Write the policy before building anything.

3
Then check

Visibility problem

The data exists but no one can see it. Requests come in via email, reports are built by hand, there's no unified view. A Qualtrics form and Tableau dashboard often solve what AI was being asked to solve.

4
AI-ready ✓

AI-ready problem

Types 1–3 are resolved. The remaining gap involves synthesis, pattern recognition, or generation at scale — the work AI genuinely does well. Now the AI conversation can happen on solid ground.

Most problems that arrive as Type 4 requests are actually Type 1, 2, or 3. That's not a failure — it's useful information, and it's a solvable thing.
